我们可以传给 v-bind:class 一个对象，以动态地切换 class。注意 v-bind:class 指令可以与普通的 class特性共存： 
data: {
    isA: false,
    isB: true
}
当 isA 和 isB 变化时，class 列表将相应地更新。例如，如果 isB 变为 true，class 列表将变为 "alert-info alert-warning"。

如果你也想根据条件切换列表中的 class，可以用三元表达式： 
v-bind:class="[classA, isB ? classB : '']
此实例始终添加alert,但是只有在isB为true时，再添加alert-success
